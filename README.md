## Setup

```sh
$ (ssh) git clone https://github.com/Raashot12/alpharithm-test.git
$ cd alpharithm-test
$ npm cache clean --force
$ npm install
# Or, npm install --legacy-peer-deps
# TODO copy .env credentials
$ npm run dev
```

#### Usage

> Local: http://localhost:3000

## Cypress Unit Testing
